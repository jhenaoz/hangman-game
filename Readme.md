## Hangman Game ##
Hangman Game using Angular and NodeJs

### First time setup ###
* Require `Node v6` and `npm v3` or bigger 
* To Handle Npm version you can use **nvm** see [here](https://github.com/creationix/nvm) for more info.

### Quick Start ###

```bash
npm install
```
```bash
npm start
```
### Running unit test ###
```bash
npm test
```
